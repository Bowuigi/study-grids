---
title: Productos genéricos, personalizados y mixtos
---

- **Productos genéricos**: Son sistemas de software que se venden en el mercado abierto a cualquier cliente que pueda comprarlos. La organización que desarrolla el software controla completamente la especificación del mismo. Ejemplos: procesadores de texto, sistemas de diseño asistido por computadora (CAD), sistemas de gestión de bases de datos.
- **Productos personalizados (o a la medida)**: Son sistemas destinados para un cliente en particular. Un contratista de software desarrolla el programa especialmente para ese cliente. La organización que compra el software generalmente desarrolla y controla la especificación, por lo que los desarrolladores deben trabajar siguiendo dicha especificación. Ejemplos: sistemas de control para dispositivos electrónicos, sistemas de control de tráfico aéreo.
- **Productos míxtos**: Son sistemas que se construyen tomando un producto genérico como base, el cual luego se adapta para ajustarse a los requerimientos específicos de un cliente. Por ejemplo: Los sistemas Enterprise Resource Planning (ERP), como el sistema SAP. En este caso, se toma un sistema grande y complejo y se adapta a una compañía al incorporar información sobre sus reglas y procesos empresariales, los reportes requeridos, etcétera.
