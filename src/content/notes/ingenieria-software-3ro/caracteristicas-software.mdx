---
title: Características de sistemas y productos de software
---

## De sistemas
- **Abstractos e intangibles**: No están restringidos por las propiedades de los materiales, las leyes físicas o los procesos de fabricación.
- **Complejidad**: Debido a la falta de restricciones físicas, pueden volverse rápidamente muy complejos, difíciles de entender y costosos de cambiar.
- **Heterogeneidad**: Cada vez más, operan como sistemas distribuidos a través de redes que incluyen diferentes tipos de computadoras y dispositivos móviles.
- **Integración**: A menudo, el nuevo software debe integrarse con sistemas legados (_legacy_) más viejos, escritos en diferentes lenguajes de programación y con otras técnicas.
- **Diversidad**: Existen muchos tipos diferentes de sistemas de software, cada uno requiriendo diferentes enfoques y técnicas de ingeniería de software.
- **Dependencia del contexto**: Las técnicas de ingeniería de software que se aplican dependen en gran medida del tipo de aplicación que se está desarrollando.

## De productos
- **Más que solo código**: Un sistema de software profesional no es solo un programa, sino que incluye toda la documentación asociada (tanto para el usuario como para los desarrolladores, incluyendo los sitios donde se descarga todo) y los datos de configuración requeridos para que los programas operen correctamente.
- **Mantenibilidad**: El software debe escribirse de tal forma que pueda evolucionar para satisfacer las necesidades cambiantes de los clientes. Es un atributo crítico debido a los entornos empresariales variables.
- **Confiabilidad y seguridad**: Incluye fiabilidad, seguridad y protección. El software no debe causar daños físicos o económicos en caso de falla, y debe ser inmune a accesos malintencionados.
- **Eficiencia**: No debe desperdiciar recursos del sistema como memoria y ciclos del procesador. Esto incluye la capacidad de respuesta y el tiempo de procesamiento.
- **Aceptabilidad**: Debe ser comprensible, utilizable y compatible con otros sistemas que usan los usuarios para quienes está diseñado.
